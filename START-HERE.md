## ðŸ†˜ Getting Help

### When Stuck on a Concept
1. **First (5-10 min)**: Try to figure it out yourself
2. **Then**: Use AI (see HOW-TO-USE-AI-WHEN-STUCK.md)
3. **Prompt format**:
   ```
   Context: I'm learning [X] in the Full Stack AI Training Program.
   Background: 15 years tech management, familiar with [Y].
   Problem: [Specific issue]
   Tried: [What I've attempted]
   Help me understand: [Specific question]
   ```

### When Code Has Errors
1. **Read error message carefully** (often tells you exactly what's wrong)
2. **Google the exact error** (someone else hit it)
3. **Use Claude for debugging** (copy-paste error + code)
4. **Check documentation** (official docs are authoritative)

### When Overwhelmed
1. **This is normal** - You're learning A LOT
2. **Take a break** (walk, coffee, fresh air)
3. **Review progress** (look at your commits, what you've built)
4. **Adjust pace** if needed (better to understand than rush)
5. **Focus on one thing** at a time
